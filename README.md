# xzboss's Blog

个人博客 & 知识库，基于 [Rspress](https://rspress.dev) 构建。

## 内容结构

- **语言** - JavaScript、CSS 等编程语言笔记
- **框架** - React、Vue 等前端框架学习笔记
- **工程化** - 设计模式、项目方案、浏览器原理
- **博客** - 技术随笔与个人思考

## 开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm build
pnpm preview
```
